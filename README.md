***Проект:* [MapsYa](https://github.com/DmitriyS2/MapsYa)**

*Задачи:* Создать Android-приложение “Карты” с возможностью:

***Проект:* [CustomTimeView](https://github.com/DmitriyS2/CustomTimeView)**

*Задачи:* Создать CustomView “Стрелочные часы” с возможностью:
- отображения правильного времени. Время отображается в момент открытия экрана,
- отображения часовой, минутной, секундной стрелок, а также анимацией их движения,
- изменения цвета циферблата, стрелок; толщины стрелок,
- изменения аттрибутов черех XML.
  
Созданы 2 варианта:

1 - отрисованный циферблат,

2 - с картинкой (jpeg) в виде циферблата часов.
*Используемые технологии:* MVVM, Animation
